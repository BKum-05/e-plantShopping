# Paradise Nursery Shopping Application

Front-end final project for the Paradise Nursery plant shop. This app includes:

- Landing page with company information and a Get Started button.
- Product listing page with categorized houseplants and Add to Cart actions.
- Shopping cart page with quantity controls, delete, total cost, and checkout message.
- Redux Toolkit state management for cart behavior.
- Routing across Home, Plants, and Cart pages.

## Tech Stack

- React + Vite
- React Router
- Redux Toolkit + React Redux

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Deploy to GitHub Pages

1. Create a public repository named e-plantShopping.
2. Update the homepage value in package.json with your GitHub username.
3. Push code to GitHub.
4. Deploy:

```bash
npm run deploy
```
